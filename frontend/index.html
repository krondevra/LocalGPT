Private local AI chat

A simple chat interface for your LM Studio model

LocalGPT is a coursework project that combines a FastAPI backend, local AI integration, authentication and chat history in one clean interface.

  • Local model connection: the backend sends requests to LM Studio.
  • Saved conversations: chats and messages are stored in SQLite.
  • Clear interface: responsive layout with keyboard-friendly interaction.

Built for a private, local workflow without unnecessary complexity.
